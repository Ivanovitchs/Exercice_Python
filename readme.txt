# Quelques exercices sur les algorithmes

Mon projet consiste à resoudre Quelques exercices en python 
qui permettra à apprendre quelques notions de base de python.
mon  workspace contient d'autres fichiers .txt qui consistent à representer 
certain graphe.
Pour lancer mes algorithmes il suffit d' executer la commande python nom du fichier.py
et pour lancer les fichiers .txt pour la representation des graphes on exécute la commande
gnuplot mon_script.gp de plus il faut s'assurer que gnuplot soit installer dans votre ordinateur
et être ajouter comme variable d'environnement.
